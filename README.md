# QQ群管理者  
QQ群管理者，一键导出QQ群成员信息，需要先进入QQ群官网！

### 功能介绍
1. 一键导出QQ群成员信息；
2. 可指定导出的群成员信息项目；
3. 支持纯文本和电子表格两种方式；
